## 二、进度检查任务

### 2.1 周进度检查

**文件：** `.claude/scheduled_tasks.json`
**时间：** 每周一 09:00
**执行命令：** 见下方

```bash
#!/bin/bash
# check-weekly-progress.sh

echo "=== AgentSkills 周进度检查 ==="
echo "日期: $(date '+%Y-%m-%d')"
echo ""

# 读取当前阶段
PHASE=$(cat ROADMAP.md | grep -A 5 "当前阶段" | head -6)
echo "当前阶段: $PHASE"

# 统计完成度
TOTAL=$(grep -c "\[ \]" ROADMAP.md)
DONE=$(grep -c "\[x\]" ROADMAP.md)
TOTAL_TASKS=$(grep -c "\[ \]\|\[x\]" ROADMAP.md)

echo "总任务数: $TOTAL_TASKS"
echo "已完成: $DONE"
echo "未完成: $((TOTAL_TASKS - DONE))"

PROGRESS=$(echo "scale=2; $DONE * 100 / $TOTAL_TASKS" | bc)
echo "进度: $PROGRESS%"

if [ $(echo "$PROGRESS < 80" | bc) -eq 1 ]; then
    echo "⚠️ 警告：进度 < 80%"
fi

if [ $(echo "$PROGRESS < 60" | bc) -eq 1 ]; then
    echo "🚨 严重警告：进度 < 60%，考虑 Pivot！"
fi

if [ $(echo "$PROGRESS < 40" | bc) -eq 1 ]; then
    echo "❌ 失败：进度 < 40%"
fi

echo ""
echo "=== 检查完成 ==="
```
