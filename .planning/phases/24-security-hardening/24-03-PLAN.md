    Create `contracts/Pausable.sol` or use OpenZeppelin:

    ```solidity
    // SPDX-License-Identifier: MIT
    pragma solidity ^0.8.20;

    import "@openzeppelin/contracts/security/Pausable.sol";
    import "@openzeppelin/contracts/access/Ownable.sol";

    contract AgentPausable is Ownable, Pausable {
        
        function pause() external onlyOwner whenNotPaused {
            _pause();
        }

        function unpause() external onlyOwner whenPaused {
            _unpause();
        }
    }
    ```

    Note: In production, pause/unpause should require governance, but for emergency
    response, owner-only allows quick action. Governance can always override.
