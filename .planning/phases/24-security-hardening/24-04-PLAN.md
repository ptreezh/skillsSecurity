<task type="automated" checkpoint="none">
  <name>Task 1: Run Slither static analysis</name>
  <action>
    Run Slither analysis on all contracts:

    ```bash
    # Install slither if needed
    pip install slither-analyzer

    # Run analysis
    slither . --solc-remaps '@openzeppelin=node_modules/@openzeppelin'

    # Generate report
    slither . --json -o docs/security/slither-report.json

    # Filter for HIGH/CRITICAL issues only
    cat docs/security/slither-report.json | jq '.results.high + .results.critical'
    ```

    Document all findings in `docs/security/slither-findings.md`.
  </action>
