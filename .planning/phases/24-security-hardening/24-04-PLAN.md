  <action>
Run Slither static analysis on all contracts and generate report.

1. Create scripts/run-security-tools.sh:
```bash
#!/bin/bash
# Security analysis script

echo "=== Running Slither ==="
slither . --json-output slither-results.json --markdown-slither slither-report.md

echo "=== Running Mythril ==="
for contract in contracts/*.sol; do
    echo "Analyzing: $contract"
    myth analyze "$contract" --solv 0.8.20 --output-dir docs/security/mythril/
done

echo "=== Security analysis complete ==="
echo "Results in: slither-results.json, slither-report.md, docs/security/mythril/"
```


- `slither-heap`: Memory issues
- `slither-knowledge`: External calls
- `slither-reentrancy`: Reentrancy vulnerabilities (mitigated by nonReentrant)

5. If issues found, fix and re-run until clean.

```markdown
# Security Report - AgentSkills v1.5

## Executive Summary
This report documents the security hardening completed for Phase 24 of the AgentSkills project.

## Vulnerabilities Fixed

### Critical (Fixed)
1. **setEffectiveReputation() allows owner to fake reputation**
   - Status: FIXED - Function removed
   - Impact: Any user could have arbitrary reputation set by owner
   - Fix: Function removed; reputation only changes through governed actions

2. **Reentrancy in unstake()**
   - Status: FIXED - ReentrancyGuard added
   - Impact: Potential recursive call attack
   - Fix: nonReentrant modifier on all state-modifying functions

### High (Fixed)
3. **Integer overflow in claimRecoverableReputation()**
   - Status: FIXED - Bounds checking added
   - Impact: Could cause incorrect recovery calculations
   - Fix: monthsElapsed capped, maxRecovery validation

### Medium (Fixed)
4. **Single point of failure (onlyOwner)**
   - Status: FIXED - GovernanceMultisig implemented
   - Impact: Single key compromise compromises entire system
   - Fix: 3-of-5 multisig for critical actions

5. **No emergency pause mechanism**
   - Status: FIXED - Pausable implemented
   - Impact: No way to halt operations during attack
   - Fix: Governance-controlled pause/unpause
```
