   ```javascript
   import { ethers } from 'ethers';
   
   // Connect to Polygon Amoy
   const provider = new ethers.JsonRpcProvider(

   );
   
   // Initialize with signer (for write operations)
   ```

   **Option A: Using ContractService (React)**
   ```javascript
   import { initContractsWithSigner, getReputation } from './services/ContractService';
   
   async function setup() {
     await initContractsWithSigner(signer, addresses);
     const rep = await getReputation(userAddress);
   }
   ```
   
   **Option B: Direct Contract Access**
   ```javascript
   import StakingManager from './abi/StakingManager.json';
   
   const stakingManager = new ethers.Contract(
     STAKING_MANAGER_ADDRESS,
     StakingManager.abi,
     provider
   );
   
   // Read
   const rep = await stakingManager.getUserReputation(user);
   
   // Write
   const tx = await stakingManager.connect(signer).stake(skillId, amount);
   await tx.wait();
   ```

   - Wallet connection (useWallet hook)
   - Network validation
   - Transaction handling (pending, success, error)
   - Event subscriptions

6. **Working with Events**
   ```javascript
   stakingManager.on('Stake', (user, skillId, amount, event) => {
     console.log(`${user} staked ${amount} on skill ${skillId}`);
   });
   
   // Cleanup
   stakingManager.removeAllListeners();
   ```

   ```javascript
   try {
     const tx = await contract.connect(signer).someFunction(params);
     await tx.wait();

       // User rejected in MetaMask
     } else if (error.code === 'INSUFFICIENT_FUNDS') {
       // Not enough MATIC for gas
     } else {
       // Other error
   ```

Create docs/faq.md:

**Structure:**

1. **Getting Started**

   **Q: What is Polygon Amoy?**
   Polygon Amoy is a Polygon testnet. You need test MATIC (not real MATIC) to interact with contracts.

   **Q: How do I get test MATIC?**
   Visit the Polygon faucet: https://faucet.polygon.technology/
   Paste your wallet address and receive test tokens.

   **Q: Why does MetaMask ask me to switch networks?**
   The dApp only works on Polygon Amoy (chainId 80002). Click "Switch Network" to continue.

2. **Wallet & Connection**

   **Q: MetaMask keeps popping up**
   Normal behavior. Every write operation (stake, unstake, register) requires wallet confirmation.

   **Q: "Transaction underpriced" error**
   Increase gas limit in MetaMask settings, or wait for network congestion to decrease.

   **Q: "Nonce too low" error**
   Reset your MetaMask activity. Settings -> Advanced -> Reset Account.

3. **Reputation & Staking**

   **Q: How is reputation calculated?**
   Reputation = base score + positive contributions - penalties. Locked amounts reduce effective reputation.

   **Q: What happens when I'm slashed?**
   Your stake is reduced, reputation is locked for 90 days. After 90 days, you can claim 5% back monthly.

   **Q: Can I recover lost reputation?**
   Yes. Submit positive contributions to enable recovery. Locked amount decreases 5% monthly.

4. **Skills & Contributions**

   **Q: What's the difference between a skill and a contribution?**
   Skills are services you offer. Contributions are actions you take on others' skills (feedback, bug reports).

   **Q: Can I delete a skill?**
   No. Skills are permanent on-chain. You can deprecate but not remove.

   **Q: Why did my skill verification fail?**
   Risk assessment by the protocol. Check skill details, ensure accuracy, and resubmit.

5. **Governance**

   **Q: What is the timelock?**
   48-hour delay between proposal approval and execution. Security measure to allow emergency intervention.

   **Q: How do I vote on proposals?**
   Go to SelfOpsPanel -> GOV tab. Connect wallet, select proposal, click For/Against.

   **Q: What's the voting threshold?**
   100 reputation points required to create a proposal. Any reputation holder can vote.

6. **Technical Issues**

   **Q: RPC errors / "Cannot connect"**
   - Check internet connection
   - Try alternative RPC: https://rpc-amoy.polygon.technology
   - Use PolygonScan RPC as fallback

   **Q: Stuck transactions**
   - Speed up in MetaMask: Select transaction -> Speed Up
   - Or cancel: Select transaction -> Cancel

   **Q: Contract calls return null**
   - Contract might not be deployed on current network
   - Check network in dApp settings

7. **Error Codes Reference**
   | Code | Meaning | Solution |
   |------|---------|----------|
   | INSUFFICIENT_FUNDS | Not enough MATIC | Get test MATIC from faucet |
   | ACTION_REJECTED | User rejected | Approve in MetaMask |
   | NONCE_EXPIRED | Transaction nonce issue | Reset account in MetaMask |
   | UNPREDICTABLE_GAS_LIMIT | Call will fail | Check contract state |

8. **Contact & Support**

   - GitHub Issues: [link]
   - Email: [link]
