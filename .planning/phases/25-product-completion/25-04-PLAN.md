    ## Getting Started

    AgentSkills is a decentralized skill marketplace where AI agent developers can:
    - Register their skills for attribution
    - Earn reputation through contributions
    - Receive rewards for verified work

    ### Prerequisites
    - MetaMask or WalletConnect compatible wallet
    - MATIC for gas fees on Polygon network
    - Basic understanding of blockchain wallets

    ---

    ## Setting Up Your Wallet

    ### MetaMask Setup

    1. Install MetaMask browser extension
    2. Create or import your wallet
    3. Add Polygon network:
       - Network Name: Polygon Mainnet
       - RPC URL: https://polygon-rpc.com
       - Chain ID: 137
       - Symbol: MATIC
       - Block Explorer: https://polygonscan.com

    4. Get some MATIC for gas fees
       - Buy from exchange and withdraw to your wallet
       - Or use Polygon Bridge to transfer from Ethereum

    ---

    ## Earning Reputation

    Reputation is the core currency in AgentSkills. You earn it by:

    ### 1. Verification Contributions
    When you verify a skill and your verification is correct:
    - +10 reputation per LOW risk skill
    - +50 reputation per MEDIUM risk skill
    - +100 reputation per HIGH risk skill

    ### 2. Bug Reports
    Submit bug reports with test reports:
    - +5 to +50 reputation based on severity
    - Negative scores for incorrect reports

    ### 3. Positive Contributions
    Any action that improves the ecosystem:
    - Quality contributions
    - Helpful reviews
    - Community support

    ---

    ## Registering a Skill

    ### Step 1: Check Your Reputation
    Different skill risk levels require minimum reputation:
    - LOW: No minimum
    - MEDIUM: 500+ reputation
    - HIGH: 2000+ reputation
    - CRITICAL: 5000+ reputation

    ### Step 2: Prepare Metadata
    Create a JSON file with skill information:
    ```json
    {
      "name": "my-skill",
      "description": "What this skill does",
      "trigger": "When to use this skill",
      "version": "1.0.0"
    }
    ```

    ### Step 3: Upload to IPFS
    Upload metadata to get CID for on-chain storage.

    ### Step 4: Register on Chain
    Use the web interface or SDK to register your skill.

    # Frequently Asked Questions

    ## General

    ### What is AgentSkills?
    AgentSkills is a decentralized marketplace for AI agent skills. It enables developers to register their skills, receive attribution for contributions, and earn rewards based on reputation.

    ### Why no token?
    We chose a token-free architecture for:
    - Regulatory compliance in all jurisdictions
    - No speculative trading
    - Focus on real utility and reputation

    ---

    ## Reputation

    ### How do I earn reputation?
    - Verify skills (+10 to +100 based on risk level)
    - Submit bug reports (+5 to +50 based on severity)
    - Quality contributions (varies)

    ### What happens if my reputation goes negative?
    You can recover negative reputation through:
    - 5% per month recovery (based on original slash amount)
    - Requires positive contributions to unlock recovery
    - Must wait at least 1 month between claims

    ### How is effective reputation calculated?
    ```
    Effective Reputation = Total Reputation - Locked Reputation
    ```

    ---

    ## Skills

    ### What are skill risk levels?
    - **LOW**: Simple skills, minimal risk (0 reputation required)
    - **MEDIUM**: Moderate complexity (500+ reputation required)
    - **HIGH**: Complex skills (2000+ reputation required)
    - **CRITICAL**: Highest risk (5000+ reputation required)

    ### Can I update my skill?
    Yes, skill owners can update:
    - Description
    - Metadata
    - Version

    ---

    ## Technical

    ### What wallet do I need?
    Any EVM-compatible wallet:
    - MetaMask
    - WalletConnect
    - Coinbase Wallet

    ### What network?
    Currently on Polygon network. Testnet: Polygon Amoy.

    ### How long do stakes lock?
    90 days from staking.

    ---

    ## Troubleshooting

    ### Transaction failed
    Common causes:
    1. **Rejected**: User cancelled in wallet
    2. **Insufficient funds**: Not enough MATIC for gas
    3. **Wrong network**: Switch to Polygon
    4. **Insufficient reputation**: Check requirements

    ### Can't see my skill
    1. Check transaction was confirmed
    2. Verify you're on correct network
    3. Try refreshing the page

    ### Reputation not updating
    1. Wait for transaction confirmation
    2. Refresh page
    3. Check if reputation is locked
    ```
