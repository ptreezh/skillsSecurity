### INFO-02: Dynamic Array Access in Tests (Non-Blocking)

**File:** `test/contracts/Attribution.test.cjs:383`

```javascript
const report = await attribution.skillTestReports(0, await attribution.testReportCount(0).then(c => c - 1n));
```

**Observation:** This pattern reads the last element of a dynamic array by computing the index dynamically.

**Severity:** Info - Pattern is valid but relies on the count being correct. If `testReportCount(0)` is 0, this would revert.

**Recommendation:** Consider adding a length check for defensive coding:
```javascript

expect(count).to.be.gt(0);
const report = await attribution.skillTestReports(0, count - 1n);
```
