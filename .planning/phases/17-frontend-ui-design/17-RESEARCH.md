# Phase 17: Frontend UI Design

## Goal

Create a complete visual design system (Design Spec) for AgentSkills frontend.

## Why

The current frontend has mock data only. Before connecting to real contracts, we need:
1. A cohesive visual identity
2. Complete component library
3. Responsive layout system
4. Dark/light mode support

## Research Areas

- [ ] Brand identity: colors, typography, logo concepts
- [ ] Component library: buttons, cards, inputs, modals
- [ ] Layout system: grid, spacing, responsive breakpoints
- [ ] Skill-specific UI: reputation tiers, risk levels, verification badges
- [ ] Web3 integration patterns: wallet connection, transaction states
- [ ] Accessibility: WCAG 2.1 compliance

## Deliverables

- UI-SPEC.md: Complete design system specification
- Figma/Sketch reference (or ASCII mockups if no design tools)
- CSS variables / design tokens
- React component patterns
