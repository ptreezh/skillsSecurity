# AgentSkills 核心价值主张

**更新日期:** 2026-05-26

---

## 定位宣言

**AgentSkills = AI Agent 技能的大众点评**

```
大众点评解决的问题: 这家餐厅好不好吃？
AgentSkills解决的问题: 这个 AI Agent 值不值得信任？
```

---

## 用户痛点

| 痛点 | 描述 | 严重程度 |
|------|------|----------|
| **技能难以搜索** | GitHub/社交媒体上分散，难以发现 | ⚠️ 高 |
| **可信度难以验证** | 没有统一标准判断技能质量 | ⚠️ 高 |

---

## 解决方案

| 痛点 | AgentSkills 解决方案 |
|------|---------------------|
| 搜索难 | 统一市场 + 分类 + 标签 + 搜索 |
| 验证难 | Auditor 审计 + 链上记录 + Reputation 系统 |

## 用户旅程

```
用户问题: "GitHub 上有 1000 个 AI Agent，但我不知道哪个可信"

AgentSkills 解决:
1. 用户搜索 "图像识别 agent"
2. 看到筛选选项:
   ├── ✅ "Verified by Auditor"
   ├── 👥 "Used by 50+ teams"
   └── ⭐ "Reputation: L4+"
3. 选择有信誉的技能
4. 订阅 / 使用 / 评价
5. Creator 获得分成 → 更多优质开发者加入
```

## 商业模式

### 付费理由

用户付费的核心理由：**为"知道哪个好"付钱**

| 价值 | 用户愿意付 |
|------|-----------|
| 节省搜索时间 | $10-50/月 |
| 保证技能质量 | $20-100/月 |
| 避免劣质技能导致的问题 | $50-200/月 |

### 订阅定价

| 服务等级 | 月费 | 包含功能 |
|----------|------|----------|
| 基础搜索 | Free | 技能列表浏览 |
| 质量保证 | $29/月 | 经验证的技能 + 基础报告 |
| 专业版 | $99/月 | 完整审计报告 + 优先支持 |
| 企业版 | $299/月 | 定制筛选 + SLA + 专属顾问 |

### 收入分配

```
用户订阅费 (例如 $99/月)
├── Creator (技能创建者): 70% = $69.3
├── Auditor (审计者): 5% = $4.95
├── Referrer (推广者): 10% = $9.9
├── Dispute Fund: 5%
└── Platform: 10% = $9.9
```

## 竞争优势

### 对比现有方案

| 方案 | 搜索 | 验证 | 归因 | 激励 |
|------|------|------|------|------|
| GitHub | ❌ | ❌ | ❌ | ❌ |
| NPM/PyPI | ⚠️ | ⚠️ | ❌ | ❌ |
| SkillHub | ⚠️ | ⚠️ | ⚠️ | ⚠️ |
| **AgentSkills** | ✅ | ✅ | ✅ | ✅ |

### 差异化要点

1. **搜索能力** - 统一的分类、标签、搜索
2. **验证体系** - Auditor 审计 + Reputation 系统
3. **链上归因** - 每次使用都被记录
4. **直接激励** - Creator 获得收入分成
